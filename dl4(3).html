陈跃芝　英语 / 县名教师工作室主持人

梅州市优秀教师、叶剑英基金优秀教师、丰顺县第二届"十佳教师"。2024 年入选广东省薪火乡村优秀教师培养资助计划（全县仅一人），同年入广东省黄春兰名教师工作室。2023 年入选市学科带头人培养对象。标签：课程思政践行者、红色实践组织者。

邓旋　道德与法治 / 德育处主任 / 县名班主任工作室主持人

初中道法高级教师，市优秀班主任，省班主任能力大赛二等奖。连续九年守毕业班，所带班级中考升学率 100%。"严而有度，爱而不纵"——把道法课上成"照亮人生的明灯"。标签：名班主任领航人、德育中枢。

杜湘瑜　物理 / 八年级班主任

初中物理一级教师，县优秀教师，主持课题获县教育教学成果奖二等奖。2011 年扎根东留至今，把抽象物理磨成"可感可触"的课堂。标签：物理一线老兵。

吴开钻　体育 / 广州支教挂职副校长

华师硕士，篮球 / 田径 / 足球国家一级裁判，广州市海珠区骨干教师。把广州"教会、勤练、常赛"模式带进东留课堂，牵头编订学校体育学科教学计划。标签：山海协作、体育活水。

卓露娟　地理 / 教学新秀

入职即担中考地理，县统考 / 市质检 / 中考优秀率均列前列，校级优秀班主任。每个孩子都是一轮明月，做"有温度的引路人"。标签：青年新锐。
